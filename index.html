The Game saves progress automatically every 1 minute.

Your save data is located in browser's local storage


Press "Save Game" button to save game, if you don't want to wait until another auto save.



ALPHA FEATURE

You can generate key to safeguard your data or continue progress on different device / computer.


Please Note that you have to generate new key if you want to include new progress after last key load


To save, press on generate key button and copy the key (Ctrl+c). Keep that key somewhere safe.


To load, paste your key (Ctrl+v) into the text box and click on "Load" button.



Press "Delete Save" button to remove save data from local storage.
